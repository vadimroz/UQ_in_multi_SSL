# UQ in Multi-SSL

![CRC Figure](crc_fig_for_git.png)

This repository contains code and data for uncertainty quantification in multi-SSL.

## Repository Structure

### Python Scripts
- **`CRC_SSL_N.py`**: CRC method for known-source count.
- **`PT_SSL_U.py`**: Implements the PT-SSL method for uncertainty quantification.

### Data
The `data/` directory contains `.npz` files organized by reverb and SNR levels:
- **`srp_dnn/`**: Data generated by SRP-DNN model.
- **`srp_phat/`**: Data generated by SSL-PHAT algorithm.

Each subdirectory contains `.npz` files for different reverb times and SNR levels.

## Usage

### Running `CRC_SSL_N.py`
To run the `CRC_SSL_N` script with configurable parameters, use the following command:

```bash
python CRC_SSL_N.py \
    --plot 1 \
    --model_type SRP_PHAT \
    --num_iterations 100 \
    --seed 1234567890 \
    --snr 15 \
    --reverb 700 \
    --speakers 3 \
    --Kmax 3 \
    --lambda_steps 1000 \
    --significance_levels 0.1 0.05
```

#### Arguments
- `--model_type`: Specify the model type (e.g., `SRP_PHAT`). Default is `SRP_PHAT`.
- `--Kmax`: Maximum speakers. Default is `3`.
- `--speakers`: Number of speakers. Default is `3`.
- `--snr`: Signal-to-noise ratio. Default is `15`.
- `--reverb`: Reverberation time in ms. Default is `700`.
- `--significance_levels`: List of significance levels. Default is `[0.1, 0.05]`.
- `--lambda_steps`: Number of thresholds. Default is `1000`.
- `--num_iterations`: Number of iterations. Default is `100`.
- `--seed`: Random seed. Default is `1234567890`.
- `--plot`: Enable or disable plotting (0 or 1). Default is `0`.

### Running `PT_SSL_U.py`
To run the `PT_SSL_U` script with configurable parameters, use the following command:

```bash
python PT_SSL_U.py \
    --dataset SYNTHETIC \
    --localization SRP_PHAT \
    --test_on_locata 0 \
    --num_iterations 100 \
    --calib_opt 200 \
    --calib_test 200 \
    --test_sets 100 \
    --seed 1234567890 \
    --snr 15 \
    --reverb 400 \
    --Kmax 3 \
    --delta 0.1 \
    --alpha_MC 0.1 \
    --alpha_MD 0.1
```

#### Arguments
- `--dataset`: Dataset type (e.g., `SYNTHETIC`). Default is `SYNTHETIC`.
- `--localization`: Localization method (e.g., `SRP_PHAT`). Default is `SRP_PHAT`.
- `--test_on_locata`: Test on LOCATA dataset (0 or 1). Default is `0`.
- `--num_iterations`: Number of iterations. Default is `100`.
- `--calib_opt`: Number of calibration optimization sets. Default is `200`.
- `--calib_test`: Number of calibration test sets. Default is `200`.
- `--test_sets`: Number of test sets. Default is `100`.
- `--seed`: Random seed. Default is `1234567890`.
- `--snr`: Signal-to-noise ratio. Default is `15`.
- `--reverb`: Reverberation time in ms. Default is `400`.
- `--Kmax`: Maximum number of speakers. Default is `3`.
- `--delta`: Delta value. Default is `0.1`.
- `--alpha_MC`: Alpha_MC value. Default is `0.1`.
- `--alpha_MD`: Alpha_MD value. Default is `0.1`.
